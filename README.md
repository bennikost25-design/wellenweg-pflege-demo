# Wellenweg Pflege – Demo

Fiktive Demo-Website für den ambulanten Pflegedienst **Wellenweg Pflege** (Erfurt).

Live: https://wellenweg-pflege-demo.vercel.app

## Entwicklung

```bash
npm install
npm run dev
```

## Qualität

```bash
npm run lint
npx tsc --noEmit
npm run build
```

Alle Unternehmensdaten sind fiktiv und dienen ausschließlich der Darstellung.
